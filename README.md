# Windsor Welfare Management System

This is a PHP-based web application for managing welfare contributions, loans, and member information for Windsor Welfare.

## Setup Instructions

1. Ensure you have XAMPP installed with Apache and MySQL running.
2. Import the `database.sql` file into your MySQL database.
3. Update the database configuration in `includes/config.php`.
4. Access the application via `http://localhost/windsor_welfare`.

## Features

- Member management
- Contribution tracking
- Loan applications and approvals
- Reports and summaries

## Project Structure

- `assets/`: Static files (CSS, JS, images)
- `includes/`: Reusable PHP files
- `pages/`: Main application pages
- `admin/`: Admin-specific pages
- `login.php`: User authentication
- `index.php`: Application entry point
